# IDP
IDP 2015 - Project Z

### Deployment
1. Get GitHub for Windows
2. Clone the repository in the GitHub application
3. Get Python 3.4.X (be sure to **add to path** during install!)
4. Open ```cmd``` and type ```pip install django```
5. Run ```runserver.bat```
6. Navigate to http://127.0.0.1:8000/
